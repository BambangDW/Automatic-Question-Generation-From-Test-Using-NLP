Requirements ::

Hardware requirements:-
1. Processor — core i3 and above
2. RAM — 4 GB or above
3. Hard Disk — 500 GB or above

Software requirements:-
1. XAMPP— Web Application Server
2. PHP— Server Side Scripting Language
3. MySQL — Database
4. Dreamweaver 8 — As a development tool
5. OS — Windows7/8/10 or Linux
6. Browser — Google Chrome
7. Pyhton - 3

Instalation ::

1). Install Xampp server in PC.
2). Install Python.
	then open python prompt and run these command

1)Copy full folder in your web directory. (C:\xampp\htdocs\  or where you installed Xampp)
2)Copy one Paragraph in Text Box and submit it.
